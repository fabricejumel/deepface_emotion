# deepface_emotion

on commence par crée le venv , la 1er fois
```bash
python3 -m venv venv_deepface_emotion_venv
```
on l'active
```bash
source venv_deepface_emotion_venv/bin/activate
```

installation des paquets la premiere fois
```bash
pip install -r requirements_deepface.txt
```

Lancement , par exemple 
```bash
python deepface_emotion.py --webcam=0
```
utiliser --help pour plus d'option
